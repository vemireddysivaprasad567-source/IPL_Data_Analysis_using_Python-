# IPL Data Analysis using Python 2007- 2024

## Overview
This project focuses on analyzing IPL (Indian Premier League) cricket match data using Python. The dashboard provides insights into team performance, toss decisions, venue analysis, finals winners, season trends, and player statistics through interactive visualizations.

---

## Objectives
- Perform Exploratory Data Analysis (EDA) on IPL datasets
- Analyze team and player performances
- Create interactive dashboards and visualizations
- Generate cricket insights using Python and Power BI

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI
- Jupyter Notebook

---

## Dataset
The dataset contains IPL match details such as:
- Teams
- Winners
- Toss decisions
- Venues
- Seasons
- Player of the Match
- Match Results

### Team Analysis
